# IndiaPulse AI

AI-powered news intelligence platform that identifies, ranks, and explains the most important news affecting India.

## Features

* News Collection
* Verification Engine
* Duplicate Detection
* India Impact Analysis
* Significance Scoring
* News Ranking
* Content Generation
* Carousel Generation
* Markdown Report Generation

## Architecture

News Sources
↓
Verification
↓
Deduplication
↓
Impact Analysis
↓
Significance Scoring
↓
Ranking
↓
Content Generation
↓
Carousel Generation
↓
Daily Report
